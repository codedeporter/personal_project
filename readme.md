# Installation and Setup Instructions

Clone this repository.  You will need node and npm installed globally on your machine.

### Run the following commands:
### For installation:
1. npm install
### To Run Test Suite
2. npm test
### To Start Server
3. npm start
### To Visit App
4. http://127.0.0.1:5500/ideas.html
